# Notus

Notus 是一款运行在懒猫微服（Lazycat MicroServer）上的私人知识库与 AI 写作助手。

## 项目结构

- `notus/`：Next.js 15 Pages Router 应用
- `lzc/`：懒猫打包与运行脚本
- `Notus_PRD.md`：技术实现规范
- `Notus_PDD.md`：产品设计文档
- `PROGRESS.md`：当前实现进度
- `CLAUDE.md` / `AGENTS.md`：仓库协作说明

## 开发

```bash
cd notus
npm install
npm run dev
```

## 构建

```bash
cd notus
npm run build
```
