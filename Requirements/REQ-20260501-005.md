## 需求标题

统一应用 PNG 图标与 LPK 打包图标。

## 背景

Header 中使用的是 `NotusLogo` 品牌图标，浏览器页签和打包图标入口未完全统一。用户希望将当前图标转换为 PNG 放在根目录，文件名固定为 `logo.png`，并让应用内 icon 与懒猫微服 LPK 共用这张图。

## 预期落地

1. 将当前 `NotusLogo` 标识导出为根目录 `logo.png`。
2. 保持 PNG 为正方形，满足 LPK 图标大小限制。
3. 保持 `lzc-build.yml` 的 `icon` 配置使用 `./logo.png`。
4. 应用内已有 icon 入口统一切换到同一张 PNG。

## 验证

1. 已按 `notus/public/notus-logo.svg` 重新导出根目录 `logo.png`，尺寸为 1024x1024，文件大小约 46KB。
2. 已新增 `notus/public/logo.png`，供前端页面和浏览器页签直接复用同一张 PNG。
3. `NotusLogo` 组件已改为渲染 `/logo.png`，Header、登录页、设置页和过渡层均会显示同一图标。
4. `notus/pages/_app.js` 已将浏览器 icon 与 Apple touch icon 改为 `/logo.png`。
5. `lzc-build.yml` 已确认继续使用 `./logo.png` 作为 LPK 图标，无需额外修改。
