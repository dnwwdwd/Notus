# REQ-20260501-007 补充 Web 分发与 LPK 打包命令

## 背景

在完成 Web + Electron 双端并存后，用户希望仓库根目录再补两个可直接执行的打包命令：一个用于生成 Web 可分发目录，一个用于执行懒猫 `.lpk` 打包，避免每次手工记忆多段命令。

## 目标

1. 提供 `npm run dist:web`，生成独立的 Web 可分发目录。
2. 提供 `npm run dist:lpk`，串起懒猫构建脚本和 `lzc-cli project build`。
3. `.lpk` 打包前自动删除同包名前缀的旧产物，符合仓库既有规则。
4. 更新根目录 README 与协作文档中的命令说明。

## 结果

- 已新增 `desktop/scripts/build-web-dist.js` 与 `desktop/scripts/build-lpk.js`。
- 已在根目录 `package.json` 增加 `dist:web` 与 `dist:lpk`。
- `dist:web` 会把 Next standalone、静态资源和 `public/` 汇总到 `web-dist/`。
- `dist:lpk` 会先删除旧 `.lpk`，再执行 `sh lzc/build-package.sh` 和 `lzc-cli project build`。
