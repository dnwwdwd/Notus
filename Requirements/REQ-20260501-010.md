# REQ-20260501-010 补充桌面端系统级快捷键与双架构打包命令

## 背景

当前桌面端虽然已经具备 Electron 壳、托管工作区和基础打包能力，但仍缺少运行时系统级快捷键、平台化快捷键提示，以及面向 macOS Intel / Apple Silicon 的清晰打包命令。

## 目标

1. 为 Electron 桌面端补充首版运行时系统级快捷键，用于唤起主窗口并直接打开搜索。
2. 保持内部快捷键配置继续使用 `Mod`，但把用户直接看到的提示文案按平台翻译成 `Command` 或 `Ctrl`。
3. 为桌面安装包补充显式的 macOS `x64`、macOS `arm64` 和 Windows `x64` 打包命令。
4. 同步更新桌面路线图、进度摘要和运行打包文档。

## 结果

- 已在 Electron 主进程中新增固定的 `CommandOrControl+K` 系统级快捷键，用于唤起主窗口并打开全局搜索。
- 已在预加载桥接和前端平台客户端中补充只读事件订阅接口，复用现有搜索弹层逻辑。
- 已在快捷键中间层新增显示格式化能力，用户可见提示会按平台显示 `Command` 或 `Ctrl`，设置页输入值继续保持 `Mod+...`。
- 已新增 `npm run dist:desktop:mac:x64`、`npm run dist:desktop:mac:arm64`、`npm run dist:desktop:win:x64` 三个显式桌面打包命令。
- 已同步更新 `README.md`、`AGENTS.md`、`CLAUDE.md`、`docs/Notus_Desktop_Roadmap.md` 和 `docs/PROGRESS.md`。
