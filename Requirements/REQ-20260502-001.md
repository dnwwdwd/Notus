# REQ-20260502-001 在快捷键设置页直接显示 macOS 与 Windows 键位

## 背景

当前快捷键设置页虽然已经支持平台化快捷键显示逻辑，但页面主体仍主要展示 `Mod+...` 的编辑值，用户需要自己阅读“填写规则”才知道 macOS 和 Windows/Linux 实际对应什么按键。

## 目标

1. 在快捷键设置页中直接展示每个快捷键对应的 macOS 与 Windows/Linux 键位。
2. 在桌面端固定系统级快捷键说明中，也同步显示两套平台实际键位。
3. 保持输入框里的编辑值继续使用 `Mod+...`，不改变现有配置格式。

## 结果

- 已在每个快捷键配置卡片中补充 `macOS` 与 `Windows / Linux` 的实际显示值。
- 已在桌面端固定系统级快捷键说明区域补充两套平台键位展示。
- 输入框、保存逻辑和底层快捷键配置格式保持不变，仍使用 `Mod+...`。
