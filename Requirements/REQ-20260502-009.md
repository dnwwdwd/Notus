# REQ-20260502-009 调整 Header 未保存按钮为红色提示

## 基本信息

- 日期：2026-05-02
- 分类：用户体验优化
- 状态：已完成
- 来源：用户请求

## 需求标题

将 Header 中未保存按钮的边框和文字改为红色。

## 背景

当前 Header 顶部的保存按钮虽然会区分“已保存 / 未保存 / 保存中”三种状态，但未保存态仍沿用主强调色，提醒强度不够。用户希望把未保存态改成更明确的红色边框和红色文字，让顶部状态一眼可见。

## 预期落地

1. Header 顶部保存按钮在“未保存”状态下使用红色文字。
2. Header 顶部保存按钮在“未保存”状态下使用红色边框。
3. “已保存”和“保存中”状态保持现有语义，不额外改动。
