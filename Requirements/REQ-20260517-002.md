# REQ-20260517-002 设置个性化页、关于页样式与桌面工作区尺寸边界

## 基本信息

- 日期：2026-05-17
- 分类：用户体验优化
- 状态：已完成
- 来源：用户提出

## 需求标题

调整设置页信息结构、关于页说明卡样式，并统一 Web / Electron 最小工作区尺寸。

## 背景

编辑器联动开关原本放在模型配置页，和模型接入不属于同一类设置；关于页版本说明卡片视觉修饰过多；同时工作区在宽度过小时需要继续同时展示侧边栏、文本编辑区和右侧聊天区。

## 本次目标

1. 新增“个性化”设置页，单独放置编辑器联动设置。
2. 个性化页不展示说明性文字，只显示功能名和开关。
3. 关于页“当前版本专注本地知识库问答、块级创作协作和桌面工作区体验。”只保留基础边框，不保留阴影、渐变、圆角等额外样式。
4. Web 端与 Electron 桌面端统一设置合理的最小宽度和高度。
5. 知识库页和创作页分栏拖拽后仍需保留文本编辑区与右侧聊天区的最小可用宽度。
