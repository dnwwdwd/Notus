# REQ-20260620-001 修正 Agent 聊天 UI 改造范围

## 分类

用户体验优化

## 背景

此前按设计稿还原 Agent 页面时，将知识库页和创作页整页替换为 Agent Workspace，导致知识库文档预览/编辑区、创作块画布、文章分片预览和批量修改预览等既有业务组件不再常驻展示。当前修正为只改聊天 UI，不替换业务主区域。

## 需求

1. 知识库页恢复旧的文档预览/编辑、来源定位、澄清抽屉、历史对话等业务区。
2. 创作页恢复旧的块画布、文章分片预览、块编辑、批量修改预览、应用/取消修改等业务区。
3. 右侧聊天 UI 按 Notus-design-draft/notus-agent.html 的聊天区、输入框、下拉、工具链、文件变更卡片和弹窗样式还原。
4. 聊天顶部去除 “Notus Agent Workspace” 文案，以及“模型配置”“搜索配置”按钮。
5. 搜索配置进入设置菜单；聊天输入框只保留联网开关和搜索商选择。
6. 设置页和引导页模型配置改为设计稿风格，LLM 保存不要求测试连通性。

## 实现摘要

1. 恢复知识库页和创作页在错误改造前的业务页面结构，并把新的 AgentWorkspace 作为右侧聊天面板嵌入原有分栏。
2. AgentWorkspace 改为无顶部标题栏的聊天面板，输入框按设计稿的附件区、textarea 区和底部控件区重做。
3. 工具链改为设计稿样式，支持执行中、完成、中止/错误、展开详情、工具输入/结果、三点动效和文件变更卡片。
4. 设置菜单新增“搜索配置”，接入现有 /api/settings/search-providers 读取和保存配置。
5. 设置页和引导页模型配置使用暖色单栏卡片风格，LLM 列表不展示密钥状态、默认配置提示或兼容协议字段。
