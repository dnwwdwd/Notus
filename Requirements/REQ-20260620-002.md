# REQ-20260620-002 移除 Agent 输入框预制问题列表

## 分类

用户体验优化

## 背景

知识库和创作页右侧 Agent 聊天输入框上方曾展示若干预制问题按钮。当前要求输入区域保持直接输入形态，不再出现预制问题列表，避免占用聊天底部空间并减少视觉干扰。

## 需求

1. 知识库页和创作页的 Agent 输入框上方不再显示预制问题列表。
2. AgentWorkspace 不再接收和渲染 suggestions。
3. 保留输入框本身的附件、联网搜索、搜索商选择、模型选择和发送/停止按钮能力。

## 实现摘要

1. 移除 AgentWorkspace / AgentInput 中的 suggestions 参数和预制问题按钮渲染逻辑。
2. 移除知识库页和创作页传给 AgentWorkspace 的 suggestions 属性。
3. 同步修复创作页因未定义 SUGGESTIONS 导致的客户端异常，bug 记录见 docs/BUG_TRACKER.md 的 BUG-20260620-001。

- npm run build:web

已完成
