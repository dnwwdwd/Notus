# REQ-20260621-003 优化 Agent Loop 工具反馈、Markdown 回复和历史导出

- 日期：2026-06-21
- 分类：功能优化
- 状态：已完成

## 背景

Agent Loop 前后端主流程已经可执行，但执行过程中的反馈和后续留档能力还不完整：工具链运行态使用刷新图标，AI 回复没有在 AgentWorkspace 中统一按 Markdown 富文本渲染，历史对话也缺少可导出的完整记录。

## 本次调整

1. 工具链运行态图标改为圆环持续旋转，不再使用刷新图标。
2. 创作页和知识库页的 AgentWorkspace AI 回复统一使用 `StreamingText` 渲染，支持 Markdown、GFM、数学公式、代码高亮和流式光标。
3. 历史对话抽屉为每条会话新增导出按钮，导出格式为 Markdown。
4. `GET /api/conversations/:id` 增加 `agent_sessions` 返回字段，用于导出同一对话下的 Agent session、工具调用日志、思考文本、快照数量和修改预览集合。

## 导出内容

导出的 Markdown 文件包含：

- 用户消息和 AI 消息正文
- 引用来源、附件、消息 meta
- 前端运行时保存的工具链步骤
- Agent session 状态、目标、授权路径和执行轮次
- `agent_run_logs` 中的工具输入、工具输出、思考文本与耗时
- 关联的文件级修改预览数据
